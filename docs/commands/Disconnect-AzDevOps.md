# Disconnect-AzDevOps

## SYNOPSIS
Disconnect from Azure DevOps

## SYNTAX

```
Disconnect-AzDevOps [<CommonParameters>]
```

## DESCRIPTION
Disconnect from Azure DevOps and remove the connection object

## EXAMPLES

### EXAMPLE 1
```
Disconnect-AzDevOps
```
